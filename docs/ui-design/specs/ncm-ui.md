# 网易云音乐大屏 UI 规格

网易云页面突出登录态个性化能力：发现、私人 FM、每日推荐、我的资产、逐字歌词和热评。共享视觉、按键、Footer Legend 和宿主安全规则见 `steam-deck-ui-rules.md`。

## 登录策略

**引导登录，不做全屏登录页**（2026-07-11 决策：登录 UI 只维护 QAM 一份，符合
DESIGN §6 职责分离——QAM 管全局控制，大屏管内容浏览）：

- 匿名可用：搜索、免费歌播放、发现页推荐歌单（本就双分支维护，无需强制登录收敛）。
- 整页需登录的内容（私人 FM、我的）用共享 `LoginGate` 空态：图标 + 提示 +
  「打开快捷菜单登录」按钮（`Navigation.OpenQuickAccessMenu()` 一键拉开 QAM），
  QAM 扫码成功经 login done 事件即时放行。
- 单点需登录的操作（每日推荐等）按错误码 `not_logged_in` 弹横幅引导。

## 全局结构

- 顶部内容首行：网易云音乐 Logo、顶层 Tabs、`L1/R1` 提示、当前播放状态徽章。
- 内容区：按当前 Tab 渲染全宽页面，不保留常驻左侧栏。
- 播放控制：无底部固定 MiniPlayer；`Start` 盲操播放 / 暂停，普通页面 `Y` 打开队列浮层，完整控制在“正在播放”或“私人 FM”页。

## 顶层页面

| 页面 | 目的 | 关键交互 |
| :--- | :--- | :--- |
| 发现 | 首页推荐、每日推荐、推荐歌单。 | `A` 播放或进入，`X` 更多操作。 |
| 私人 FM | NCM 核心沉浸电台。 | `Y` 红心，`X` 垃圾桶 / 下一首，`Start` 播放 / 暂停。 |
| 搜索 | 找歌和热搜入口。 | 搜索框 `A` 唤键盘，热搜词 `A` 直接搜索。 |
| 我的 | 账号资产。 | 二级 Tab：我喜欢、排行、创建歌单、收藏歌单、云盘。 |
| 正在播放 | 逐字歌词、翻译、热评。 | `X` 在歌词 / 热评间切换。 |

## 发现页

结构：

- 顶部 Banner 横向轮播。
- 每日推荐 30 首入口。
- 推荐歌单网格。
- 顶层 Tab 行右侧状态徽章显示当前播放，不参与焦点。

交互：

- 每日推荐按 `A` 可直接入队播放。
- 歌单卡片按 `A` 进入详情，按 `X` 打开菜单。
- 网格使用 `MAINTAIN_X`。

## 私人 FM

私人 FM 是专属播放器页面，不使用全局队列的完整未来列表。

规则：

- 没有上一首。
- 三个主操作：红心、播放 / 暂停、垃圾桶。
- `Y` 红心。
- `X` 垃圾桶并切下一首。
- `Start` 播放 / 暂停。
- 当前歌曲快结束或队列不足时补拉 FM 歌曲。
- Footer Legend 必须显示“红心 / 垃圾桶”的局部覆盖语义，而不是普通队列语义。

## 搜索页

结构：

- 搜索框。
- 热搜榜侧栏。
- 搜索结果列表。
- 二级分类：单曲、歌单。

交互：

- 搜索框按 `A` 唤起软键盘。页面无底部固定栏，避免键盘压缩视口时遮挡内容。
- 未输入时显示热搜；选中热搜词按 `A` 直接搜索。
- 结果列表按 `A` 播放或进入，按 `X` 打开菜单。
- 如果顶层页面使用 `L1/R1`，二级分类可使用 `L2/R2`。

## 我的

采用 Steam 官方库页面范式：全宽二级 Tab 行 + 全宽内容区。

结构：

- 不重复大头像 / 大横幅；账号信息、退出登录、每日签到放 QAM 或设置浮层。
- 二级 Tab 带计数徽章：我喜欢、听歌排行、创建歌单、收藏歌单、我的云盘。
- “我喜欢”Tab 使用全宽高密度歌曲列表。
- 创建 / 收藏歌单 Tab 使用全宽封面网格。
- `L2/R2` 或十字键左右切二级 Tab，十字键向下进入内容区。

退出登录必须弹确认框，避免误触。

## 正在播放和热评

布局：

- 左侧三分之一：大封面、歌名、歌手。
- 右侧三分之二：逐字歌词、翻译或热评。
- 控制组提供进度、音量、上一首 / 下一首、播放模式。私人 FM 不显示上一首。

交互：

- 默认展示歌词。
- `X` 切换歌词 / 热评。
- 热评列表中 `Y` 可作为点赞快捷键；不可用时不显示提示。
- 十字键上下手动浏览歌词或评论，`B` 返回自动跟随或歌词视图。

## 验收

- 未登录时需登录页面显示 LoginGate 引导，不渲染空数据。
- 纯手柄能完成发现播放、FM 红心 / 垃圾桶、搜索、队列浮层、歌词 / 热评切换。
- FM 页面无上一首入口。
- 软键盘弹出时没有底部播放条被顶起。
- 登录过期、二维码过期、接口空列表、不可播歌曲都进入可恢复错误态。
