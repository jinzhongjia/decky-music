# 网易云音乐大屏 UI 规格

网易云页面突出登录态个性化能力：发现、私人 FM、每日推荐、我的资产、逐字歌词和热评。共享视觉和手柄规则见 `steam-deck-ui-rules.md`。

## 登录策略

技术上，部分免费歌曲可以匿名获取播放 URL；产品层 P3 UI 仍采用强制登录：

- 未登录时 `/music` 只显示扫码登录页。
- 登录后再显示发现、FM、搜索、我的、播放页。
- 这样可以减少匿名 / 登录两套状态分支，并保证日推、FM、红心、云盘等功能可用。

这只是 UI 产品决策，不改变 provider 协议事实。

## 顶层页面

| 页面 | 目的 | 关键交互 |
| :--- | :--- | :--- |
| 登录 | 获取账号态。 | 默认焦点在刷新二维码；`A` 刷新，`B` 返回。 |
| 发现 | 首页推荐、每日推荐、推荐歌单。 | `A` 播放或进入，`X` 更多操作。 |
| 私人 FM | NCM 核心沉浸电台。 | `Y` 红心，`X` 垃圾桶 / 下一首，`Start` 播放 / 暂停。 |
| 搜索 | 找歌和热搜入口。 | 搜索框 `A` 唤键盘，热搜词 `A` 直接搜索。 |
| 我的 | 账号资产与退出登录。 | 红心、排行、云盘、签到、歌单。 |
| 正在播放 | 逐字歌词、翻译、热评。 | `X` 在歌词 / 热评间切换。 |

## 登录页

状态：

- 等待扫码。
- 已扫码，等待手机确认。
- 二维码过期，可按 `A` 刷新。
- 登录成功后自动进入发现页。

登录页不显示普通内容 tab，避免未登录状态下的空数据分支。

## 发现页

结构：

- 顶部 Banner。
- 每日推荐 30 首入口。
- 推荐歌单网格。
- 底部播放条可通过 `View` 直达。

交互：

- 每日推荐按 `A` 可直接入队播放。
- 歌单卡片按 `A` 进入详情，按 `X` 打开菜单。
- 网格使用 `MAINTAIN_X`。

## 私人 FM

私人 FM 是专属播放器页面，不使用全局底部播放条。

规则：

- 没有上一首。
- 三个主操作：红心、播放 / 暂停、垃圾桶。
- `Y` 红心。
- `X` 垃圾桶并切下一首。
- `Start` 播放 / 暂停。
- 当前歌曲快结束或队列不足时补拉 FM 歌曲。

## 搜索页

结构：

- 搜索框。
- 热搜榜侧栏。
- 搜索结果列表。
- 二级分类：单曲、歌单。

交互：

- 搜索框按 `A` 唤起软键盘。
- 未输入时显示热搜；选中热搜词按 `A` 直接搜索。
- 结果列表按 `A` 播放或进入，按 `X` 打开菜单。
- 如果顶层页面使用 `L1/R1`，二级分类可使用 `L2/R2`。

## 我的

结构：

- 账号横幅：头像、昵称、会员标识、退出登录。
- 快捷入口：我喜欢的音乐、听歌排行、我的云盘、每日签到。
- 创建歌单和收藏歌单分区。

退出登录必须弹确认框，避免误触。

## 正在播放和热评

布局：

- 左侧三分之一：大封面、歌名、歌手。
- 右侧三分之二：逐字歌词、翻译或热评。

交互：

- 默认展示歌词。
- `X` 切换歌词 / 热评。
- 热评列表中 `Y` 可作为点赞快捷键；不可用时不显示提示。
- 十字键上下手动浏览歌词或评论，`B` 返回自动跟随或歌词视图。

## P3 API 缺口

| 能力 | UI 用途 |
| :--- | :--- |
| Banner | 发现页顶部推荐。 |
| 每日推荐 | 发现页核心入口。 |
| 推荐歌单 | 发现页网格。 |
| 私人 FM | FM 页面歌曲流。 |
| FM 垃圾桶 | FM 页面跳过并反馈不喜欢。 |
| 红心 / 取消红心 | FM、搜索结果、我的资产。 |
| 热搜 | 搜索页未输入状态。 |
| 用户歌单 | 我的页面歌单分区。 |
| 听歌排行 | 我的页面快捷入口。 |
| 云盘 | 我的页面快捷入口。 |
| 每日签到 | 我的页面快捷入口；可后置。 |
| 喜欢列表 | 搜索结果红心态、我的页面。 |
| 逐字歌词和翻译 | 正在播放页。 |
| 歌曲评论 | 热评切换页。 |

## 验收

- 未登录时只显示登录页，不渲染空发现 / 我的页面。
- 纯手柄能完成登录刷新、发现播放、FM 红心 / 垃圾桶、搜索、歌词 / 热评切换。
- FM 页面无上一首入口。
- 登录过期、二维码过期、接口空列表、不可播歌曲都进入可恢复错误态。
